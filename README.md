To anyone who wants to edit or run the code in the future here are the steps :

1 - Git Clone the project in your laptop  
2 - Run `npm install` in your VS Code  
3 - Run `npm run dev` in your VS Code  
4 - Make the changes you want  
5 - `git push` the changes  
6 - Run these in your VS Code Terminal :  
    a) `npm run build`  
    b) `npm run deploy`
